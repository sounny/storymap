## Tutorial Overview

This tutorial covers:
1. Setting up your Story Map
2. Adding content

4. Using the Sidecar (immersive element)
5. Creating maps within Story Maps
6. Map Tours and Express Maps
7. Adding other media types
8. Preview and Publish

## 1. Accessing Story Maps

### Two ways to access:
- **From ArcGIS Online:** Click the "waffle" menu in the top bar
- **Direct URL:** Visit [storymaps.arcgis.com](https://storymaps.arcgis.com) and sign in

### Helpful Resources
Before starting, explore the resources in the bottom left of the StoryMaps homepage:
- **Explore other people's stories** - Great for inspiration
- **Tutorials** - How-to guides
- **Planning stories** - Strategy and structure advice

---

## 2. Creating a New Story Map

1. Click the green **"New Story"** button
2. Choose **"Start from scratch"** (other options available)

### Initial Setup
- **Title:** Give your story a name
- **Description:** Add a short description
- **Cover Image:** Optional background image
- **Design Options:**
  - Cover layout (various styles available)
  - Navigation (highly recommended - allows readers to navigate chapters)
  - Theme (choose colors that match your topic)

> ğŸ’¡ **Tip:** The navigation feature is highly recommended as it allows readers to navigate through different chapters.

---

## 3. Adding Content

### Creating Chapters
1. Click the **+** button
2. Select **Text**
3. Choose **Heading 1** from the dropdown for chapter titles
4. Chapter automatically appears in the navigation bar

### Adding Body Text
1. Click **+** button
2. Select **Text**
3. Choose **Paragraph** for regular content

### Text Hierarchy
- **Heading 1** = New chapter/section (appears in navigation)
- **Heading 2** = Sub-section (doesn't create new navigation item)
- **Paragraph** = Body text

---

## 4. Adding Maps from ArcGIS Online

### To add an existing map:
1. Click **+** button
2. Select **Map**
3. Choose your prepared map
4. Adjust settings:
   - Toggle layers on/off
   - Pan and zoom the map view

### Map Sizing Options
- **Small** - Compact view
- **Medium** - Default size
- **Large** - Full screen width

> ğŸ’¡ **Tip:** Use the **six dots** handle to drag and reposition content. Maps can sit alongside text for a snug layout.

---

## 5. Using the Sidecar (Immersive Element)

The Sidecar is an immersive scrolling element with a media panel.

### To add a Sidecar:
1. Click **+** button
2. Select **Sidecar**
3. Choose version (e.g., "Docked")

### Adding content to Sidecar:
- Add text (use Heading 2 to avoid creating new navigation sections)
- Add images
- Click the **+** button within the sidecar to add more slides/elements

---

## 6. Map Tours

Map Tours showcase a series of locations with associated content.

### Option 1: Start with a Feature Service
Use an existing layer from ArcGIS Online:
1. Click **+** â†’ **Map Tour**
2. Select **Feature Service**
3. Choose your point layer
4. Points and names are automatically populated

> âš ï¸ **Note:** Images won't appear unless photos are attached to the point features in the original data.

### Option 2: Start from Scratch
1. Click **+** â†’ **Map Tour**
2. Select **Start from scratch**
3. Choose layout (e.g., "Explorer" with list view)
4. For each tour point:
   - Click "Untitled tour point"
   - Add a **title**
   - Add a **location** (zoom to area on map)
   - Set the **zoom level**
   - Add an **image** (optional)

## 7. Express Maps (Built-in Map Creator)

Create simple maps directly within Story Maps.

### To create an Express Map:
1. Click **+** â†’ **Map**
2. Select **New Express Map**

### Drawing Tools Available:
- Points
- Numbered points
- Lines
- Freehand drawing
- Annotations (text labels)
- Arrows

### Adding Points:
1. Zoom to desired location
2. Select point icon
3. Click on map to place point
4. Add name/title
5. Choose style (can use custom symbols)
6. Click **Done**

### Adding Annotations:
1. Click the **Big T** icon
2. Type your annotation text
3. Position as needed

### Adding Arrows:
1. Select arrow tool
2. Draw arrow
3. Hover to see handle for repositioning
4. Drag to trash bin to delete

---

## 8. Map Actions (Interactive Zoom)

Map actions allow readers to click buttons that zoom to specific locations.

> âš ï¸ **Note:** Map actions work best within a **Sidecar** element.

### To add Map Actions:
1. Create a Sidecar with your Express Map
2. Add text that will become the clickable link
3. Select the text
4. Add map action â†’ "Zoom to [location name]"
5. Set the zoom level and view
6. Repeat for other locations

**Result:** Readers can click on each point name and the map automatically zooms to that location.

---

## 9. Adding Other Media

### Video
Two options:
1. **Upload** - From a file on your computer
2. **Embed** - From the internet (YouTube, Vimeo, etc.)

After adding, resize as needed.

### Audio
- Download audio files (e.g., soundscapes)
- Upload directly to Story Map
- Great for background ambiance while readers browse

### Image Galleries
1. Click **+** â†’ **Image Gallery**
2. Select multiple photos at once for easier uploading
3. Wait for photos to load
4. Rearrange as needed
5. Click **settings** to choose display style:
   - **Film strip** - Readers can flick through photos

---

## 10. Preview Your Story

Before publishing, always preview:

1. Click the **Preview** button (top of editor)
2. Story Maps opens a reader-view interface
3. Scroll through entire story
4. Test all interactive elements:
   - Navigation works
   - Map actions function
   - Media displays correctly
5. Make any final edits

---

## 11. Publishing Your Story Map

### To Publish:
1. Click **Publish** button (top right)
2. Set sharing level:
   - **Private** - Only you can view
   - **Organization** - Anyone in your organization
   - **Public** - Everyone

### Additional Options:
- Share with specific **groups** in your organization
- **Allow duplication** - Others can copy your story

> ğŸ’¡ **Tip on Duplication:** Great for brands - colors, logos, themes, and structures can be easily replicated. Also helpful for giving others a head start on similar narratives.

---

## âš ï¸ Important: Sharing Privileges

**Critical reminder:** Any content stored in ArcGIS Online that is used in your Story Map must have the **same sharing privileges** as the Story Map itself.

- If your Story Map is public, your maps/layers must also be public
- You will be prompted to update sharing levels during publishing if there's a mismatch
