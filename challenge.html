Challenge Requirements

‚úÖ

Must Include

  • Professional Cover

    High-quality image, compelling title, and descriptive subtitle

  • Clear Introduction

    Sets context, explains "why this matters," and previews content

  • At Least One Map

    Express Map or embedded Web Map with multiple features

  • Multiple Content Blocks

    Minimum 6 content blocks with variety (text, images, maps)

  • One Immersive Section

    Sidecar (with Map Actions for guided navigation), Slideshow, or Map Tour

  • Conclusion with Call-to-Action

    Summarize key points and guide readers to next steps

‚≠ê

Should Include

  • Custom Theme

    Thoughtfully chosen colors that match your topic

  • Accessibility Features

    Alt text on all images, good color contrast

  • Mobile Optimization

    Tested and looks good on phone screens

  • External Links

    References to sources or related resources

üåü

Bonus Points

  • Embedded Web Map

    Create a custom map in ArcGIS Online with multiple layers

  • Map Tour from Feature Service

    Import existing point data from ArcGIS Online for your tour

  • Map Actions in Sidecar

    Auto-zoom to different locations as readers scroll through slides

  • Video Content

    Include relevant video (self-recorded or embedded from YouTube)

  • Audio Narration

    Add audio to enhance the storytelling experience

  • Image Gallery

    Use the filmstrip layout to showcase multiple related images

  • Express Map Annotations

    Use arrows, text labels, and numbered points for detailed maps

Evaluation Criteria

Your StoryMap will be evaluated on these dimensions:

Content Quality

30%

Is the information accurate, well-researched, and clearly communicated? Does it tell a compelling story?

Map Integration

25%

Are maps used effectively to support the narrative? Do they add value beyond what text alone could convey?

Visual Design

20%

Is the design cohesive and professional? Are images high-quality and properly sized?

User Experience

15%

Is the story easy to follow? Does it work well on different devices? Is navigation intuitive?

Technical Execution

10%

Are all features working correctly? No broken links, images, or map errors?
